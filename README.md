# Analog Clock (HTML + CSS + JavaScript)

A small **frontend learning project** that recreates a real-time **analog clock** using HTML, CSS and JavaScript.

This project was built to practice **layout positioning, visual styling and dynamic UI updates**.

---

## Features

- Real-time analog clock
- Rotating hour, minute and second hands
- Clean layout using CSS positioning
- JavaScript time-based updates

---

## Tech Stack

- HTML
- CSS
- JavaScript

---

## Run Locally

No build tools are required.
Simply open:
```bash
index.html
```
in your browser.

---

This project focuses on practicing:
 - CSS layout and positioning
 - Basic DOM manipulation
 - Working with JavaScript timers
 - Updating UI elements dynamically
